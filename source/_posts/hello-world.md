## 快速运行指南
### 创建新文章
``` bash
$ hexo new "My New Post"
```

### 本地启动预览
``` bash
$ hexo server
```

### 生成静态网站文件
``` bash
$ hexo generate
```

### 一键部署到线上
``` bash
$ hexo deploy
```
