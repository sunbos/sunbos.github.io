欢迎来到 [Hexo](https://hexo.io/)！这是你的第一篇文章。想了解更多信息请查阅 [官方文档](https://hexo.io/docs/)。如果在使用的过程中遇到任何问题，你可以在 [常见问题解答](https://hexo.io/docs/troubleshooting.html) 中寻找答案，也可以在 [GitHub](https://github.com/hexojs/hexo/issues) 上提问。

## 快速运行指南
### 创建新文章
``` bash
$ hexo new "My New Post"
```

### 本地启动预览
``` bash
$ hexo server
```

### 生成静态网站文件
``` bash
$ hexo generate
```

### 一键部署到线上
``` bash
$ hexo deploy
```
